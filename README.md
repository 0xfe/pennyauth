# Pennyauth

Pennyauth is a captcha replacement that uses micropayments. Try the demo on https://pennyauth.com.

Also see the [pennyauth-server](https://github.com/0xfe/pennyauth-server) repository.

# Building

```
npm i
npm run clean
npm run build
```

# Hacking

Start a development server:

```
npm i
npm start
```
